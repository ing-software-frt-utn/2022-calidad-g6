-------------------------- TFI Inspeccion de Calzado ---------------------------------

******** Solución Cliente ********

- Desarrollada en Lenguaje C# para SO Windows, en el IDE Visual Studio 2019.

******** Solución Servidor ********

- Desarrollada en Lenguaje C# para SO Windows, en el IDE Visual Studio 2019.
- Hostea un servicio de servidor en una aplicación de consola.
- Se utilizó Entity Framework 6.4 como ORM para gestionar la conexión con la DB.
- Se utilizó Unity para gestionar la inyeción de dependencias y el patron factoría.
- Se utilizó MSTest y Moq para la gestion de pruebas en el proyecto.

------------------ Requisitos ---------------------

- Para poder utilizar el proyecto tal cual esta, se debe tener instalado SQL Server en el ordenador.
- La solución Servidor debe ejecutarse como Administrador para que Windows permita alojar el servicio.
 
------------------ Consejos -----------------------

- Si no desea instalar SQL Server, debera cambiar las cadenas de conexión en los archivos AppConfig.
